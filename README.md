 Tecnologías Utilizadas
Vue.js 3 (Composition API): Framework principal utilizado para estructurar la aplicación de forma modular y manejar la reactividad del sistema (navegación entre vistas, estados de sesión y validaciones de formularios).
Tailwind CSS: Framework de diseño utilizado para maquetar la interfaz estética y minimalista de la clínica mediante clases utilitarias, garantizando un diseño adaptable y animaciones fluidas.
Vite: Herramienta de construcción utilizada para compilar el proyecto de manera eficiente, optimizar el manejo de activos (imágenes de la clínica) y proveer un entorno de desarrollo rápido.
